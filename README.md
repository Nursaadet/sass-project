# Sass Project

This project is a website built entirely using **Sass**. It showcases various Sass features and demonstrates how they can be applied in real-world front-end development.

## 🔧 Features

- Full usage of **Sass** features  
- Utilization of `@extend`, `@mixin`, and other Sass functionalities  
- Responsive and clean design built with **pure CSS** enhanced by Sass
